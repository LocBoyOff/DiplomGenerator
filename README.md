# Генератор дипломов

GUI-приложение для генерации PDF-дипломов из Excel + шаблон PPTX.

## Установка
pip install -r requirements.txt

## Запуск
python diploma_generator.py

## Использование
1. Выбери Excel с данными (столбцы: ФИО, Дата и т.д.).
2. Выбери шаблон PPTX с плейсхолдерами {ФИО}, {ДАТА}.
3. Выбери папку для PDF.
4. Сопоставь поля и запусти!
